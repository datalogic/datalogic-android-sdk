This package contains sub packages capable of configuring and controlling completely the Datalogic device.

See the following classes: {@link com.datalogic.device.info.SYSTEM}, {@link com.datalogic.device.input.KeyboardManager}, {@link com.datalogic.device.input.TouchManager}, {@link com.datalogic.device.location.LocationManager}, {@link com.datalogic.device.nfc.NfcManager}, {@link com.datalogic.device.notification.LedManager} and {@link com.datalogic.device.power.PowerManager}
